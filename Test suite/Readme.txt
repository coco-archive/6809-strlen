The test suite is designed to be used with a TRS-80 Color Computer 1, 2 or 3
with at least 16K of RAM and Disk Extended Color BASIC.  It can also be used
with XRoar and MAME/MESS.

1.  Insert the disk STRLEN.DSK
2.  Type RUN "TEST"
